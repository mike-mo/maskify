Maskify — Privacy Policy

Last updated: April 2026

What the extension does

Maskify reads the HTML content of the web page you are currently viewing in order to detect email address patterns. When you click the Maskify button, detected addresses are replaced with randomly generated placeholder addresses using the example.com domain (reserved for documentation per RFC 2606).

Data collection

Maskify does not collect, record, or store any personal data. No information about you, your browsing activity, or the email addresses found on any page is retained beyond the lifetime of the current browser tab.

Data transmission

Maskify makes no network requests. All processing happens entirely within your browser. No data is sent to any server, third party, or external service.

Third-party sharing

Because no data leaves your browser, no data is shared with any third party.

Permissions

Maskify requests the tabs permission solely to send a message from the extension popup to the content script running in the active tab. This permission is not used to read tab URLs, titles, or any other tab metadata.

Contact

Questions or concerns about this policy can be raised via the GitHub Issues page.
