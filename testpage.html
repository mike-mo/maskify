Maskify Test Page

Click the Maskify button in the extension popup, then verify each scenario below matches its expected result.

1. Plain Text (no link)

Single email in a sentence

For questions, contact support@acme.com directly.

Expected: email replaced with a fake @example.com address
Multiple different emails in one paragraph

Reach Alice at alice@company.com or Bob at bob@company.com, or copy both via team@company.com.

Expected: three different fake addresses — all distinct from one another

2. Mailto Links

Standard mailto href — display text matches href

info@example.org

Expected: both the visible text and the href= value are replaced with the same fake address
Mailto href with a different display label

Contact Billing

Expected: href= value is replaced; visible text "Contact Billing" is unchanged

3. Consistency — Same Email, Multiple Locations

Same address repeated in different contexts

Primary contact: repeat@test.com

Send invoice to: repeat@test.com

CC list includes repeat@test.com as well.

Expected: all three occurrences show the same fake address (consistent mapping)

4. Email Format Variations

Plus-addressing / subaddressing

user.name+filter@newsletter.io

Expected: entire address including + tag replaced
Subdomain sender

alerts@mail.notifications.bigcorp.com

Expected: replaced normally
Dots in local part

first.last.middle@domain.co.uk

Expected: replaced normally

5. Tabular Data

Emails inside a table
NameEmailRole
Carolcarol@hr.example.orgManager
Davedave.smith@ops.example.orgEngineer
Evecarol@hr.example.orgAnalyst
Expected: Carol and Eve share the same fake address (same real email); Dave gets a different fake address

6. Non-Email @ Symbols (Should Not Change)

Social media handles and code snippets

Follow us @maskify on Twitter. In bash: echo $USER@$(hostname)

Expected: neither @maskify nor the shell snippet is treated as an email address

7. Email in an Input Field

Input with email as placeholder and value

Expected: both the value= and placeholder= attributes are replaced
